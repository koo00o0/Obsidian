## 단축키 

검색 : Ctrl + shift + F 
뒤로 : Ctrl + ← (설정에서 단축키 설정 default : Ctrl+Alt+←)
앞으로 : Ctrl + → (설정에서 단축키 설정 default : Ctrl+Alt+→)
줄 위아래 바꾸기 : Ctrl + ↑ or ↓  (설정에서 단축키 설정)
템플릿 삽입 : Ctrl + T  (설정에서 단축키 설정)
쓰기, 읽기 전환 : Ctrl + E

메모 내 검색 :  Ctrl + F
명령어 팔레트 열기 : Ctrl + P
빠른 파일 열기 : Ctrl + O
새 메모 만들기 : Ctrl + N
